# EthicGuard — Issue Tracker

Public **bug reports** and **feature requests** for [EthicGuard](https://ethicguard.ai) — AI acceptance-criteria QA for Jira, on the Atlassian Marketplace.

- 🐞 **Report a bug** → [new bug report](../../issues/new?template=bug_report.yml)
- 💡 **Request a feature** → [new feature request](../../issues/new?template=feature_request.yml)
- 📋 **See what's already reported** → [browse open issues](../../issues?q=is%3Aissue)

> This repository intentionally contains **no source code** — it exists only as the public issue tracker for the EthicGuard app.

For private matters (billing, security, your data), email **admin@ethicguard.ai** or use the [contact page](https://ethicguard.ai/contact). EthicGuard is zero-retention — please don't paste Jira content or credentials into public issues.
